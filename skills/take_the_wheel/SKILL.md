# NOME DA SKILL: Take the Wheel (Assumir o Controle)
# OBJETIVO: Mudar o comportamento do agente de reativo para PROATIVO para refinar ideias de produtos digitais antes de enviá-las ao banco de dados do Streamlit.

# REGRAS DE EXECUÇÃO:
1. Quando o usuário disser "Take the wheel" ou "Vamos focar nisso", pare de sugerir e comece a perguntar.
2. Faça APENAS UMA pergunta por vez. Nunca envie uma lista de perguntas.
3. Sugira opções para facilitar a resposta do usuário (A, B ou C).
4. A pergunta principal que bloqueia o avanço deve SEMPRE estar na última linha, em LETRAS MAIÚSCULAS.
5. Quando tiver informações suficientes (Nicho, Dor, Preço Sugerido e Formato), encerre o processo e envie a ideia formatada para o banco de dados (Status: Pendente de Aprovação no Painel).
